# Tokenable

Marketing site for Tokenable — Next.js (App Router), Tailwind, deployed on Netlify.

## Local dev

```bash
npm install
npm run dev
```

Set `NEXT_PUBLIC_FORMSPREE_ENDPOINT` in `.env` for the contact form (Formspree form URL).

## Build

```bash
npm run build
```

Uses `@netlify/plugin-nextjs` (see `netlify.toml`).
